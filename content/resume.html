Summary

I am an applied data scientist with a passion for developing innovative tools and analyses using advanced analytical techniques to help solve complex business problems. Since 2016, I’ve worked in the health insurance industry performing analyses on healthcare claims and (more recently) Marketing and Sales campaigns. I offer a rare blend of technical and communication skills, which helps to maximize my analyses and the potential impact. My solutions typically involve data mining, visualizations and dashboards, and predictive models.

I love working with the R programming language and frameworks like the tidyverse, caret, htmlwidgets, text2vec, and rmarkdown. At work and in my spare time, I enjoy developing R packages when I learn new techniques that can be consolidated into a series of functions. I consider myself a functional programmer who uses a series of pure functions to chain together an analysis. You can find some of my open source projects on my GitHub, including my outsiders package which implements a selection of outlier detection techniques in R.

I have some experience with Python, namely for computer vision applications using OpenCV and Tesseract. I am interested in continuing to learn more Python for machine learning, seeing as the top cloud computing companies (e.g. AWS) offer more support for Python than R. Other tools I use frequently include SQL, Alterx, Git, and Jira.

Finally, I enjoy teaching and mentoring. For the last few years, I’ve developed and led several trainings on R programming, applied statistics, and predictive modeling. This year, I am formally mentoring an intern for 10 weeks. He and I are working on some text mining methods.

The system has several applications, including duplicate detection and automatic verification of information. Currently, simtext is used as a command line application, but future plans are to implement simtext as an Alteryx workflow with a lightweight UI for self-service utilization.

  • Creator of isotext, an information retrieval system that utilizes predictive modeling to extract customer contact information from eletronic PDF documents. The system was built on creative feature engineering and multi-class classification using XGBoost. The functions for feature engineering are exposed via an R package, and the classifier is deployed to SQL Server Machine Learning Services for batch predictions. The classifier performs at 99.8% accuracy after 10-fold cross-validation, effectively eliminating the need for humans to spend hours searching for the information.
